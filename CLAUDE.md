## Project: AI Tools Expense Tracker

> Full architecture, data flow, and dev rules are in [AGENTS.md](./AGENTS.md) - read it first.

Python + Next.js expense tracker. Scans Gmail for invoices, builds Excel + Next.js dashboard, and deploys to GitHub Pages.

### Commands
```bash
python build_report.py                 # build Excel + docs/data.json
python auto_invoice.py                 # run Gmail invoice scan manually
python rebuild_graphify.py             # refresh graphify-out/*

cd dashboard-web && npm ci
cd dashboard-web && npm run dev
cd dashboard-web && npm run build      # prebuild: sync-data.mjs, postbuild: export-to-docs.mjs
cd dashboard-web && npm run lint
cd dashboard-web && npm run typecheck
```

### Architecture
```
build_report.py          - merges both data sources -> Excel + docs/data.json
auto_invoice.py          - Gmail scanner (runs daily 08:00 via Task Scheduler)
MANUAL_TRANSACTIONS      - auto-imported/manual code entries written by auto_invoice.py
manual_receipts.json     - dashboard manual entries written via GitHub Direct API
dashboard-web/           - Next.js dashboard; build syncs docs/data.json into public/data.json
docs/                    - GitHub Pages root (auto-generated, never edit manually)
```

### Verification
- After Python changes: `python build_report.py`
- After TypeScript changes: `cd dashboard-web && npm run build`
- If you touched dashboard types or client logic: `cd dashboard-web && npm run lint && npm run typecheck`

### Key Gotchas
- `build_report.py` merges `MANUAL_TRANSACTIONS` + `manual_receipts.json`; never add the same receipt to both sources
- Dashboard manual imports use `dashboard-web/public/manual-import-config.json` in `github-direct` mode and write directly to GitHub, which triggers `.github/workflows/rebuild-manual-imports.yml`
- Branch is `master`; CI mirrors changes to `main`
- Never edit `docs/data.json` directly - generated by `build_report.py`
- Never commit `dashboard-web/public/data.json` - generated during dashboard prebuild and ignored by git
- Windows absolute paths in Python must be guarded with `os.environ.get("CI")`
- CapCut uses hardcoded fallback `CAPCUT_FALLBACK_ILS = 49.90` in `auto_invoice.py`; update if price changes
